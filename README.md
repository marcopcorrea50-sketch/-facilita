# FacilITa — Sistema de Instrução de Trabalho

Sistema digital PWA para criação e gestão de Instruções de Trabalho.
VALE S.A. · Serra Sul S11D · GER Integridade Estrutural

## Como publicar no GitHub Pages

1. Crie um repositório no GitHub chamado `facilita`
2. Faça upload de TODOS os arquivos desta pasta
3. Vá em **Settings → Pages → Branch: main → / (root) → Save**
4. Acesse: `https://SEU-USUARIO.github.io/facilita`

## Como instalar como app no celular

### Android (Chrome)
1. Abra a URL no Chrome
2. Toque nos 3 pontinhos ⋮
3. "Adicionar à tela inicial"
4. Confirme → aparece o ícone do FacilITa

### iPhone (Safari)
1. Abra a URL no Safari
2. Toque no botão compartilhar ↑
3. "Adicionar à Tela de Início"
4. Confirme → aparece o ícone do FacilITa

## Funciona offline
Após o primeiro acesso, o app funciona sem internet.
